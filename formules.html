Code

Stage accéléré en 5 jours 490€

20% de réduction pour les étudiants

Sur une période de 5 jours(de 9h à 12h et de 13h à 17h)soit pendant 35 heures, K6 autoécole vous propose la formation au code de la route en accéléré avec un enseignant vous accompagnant tout au long du stage. Rapide et efficace, ce stage code vous prépare en très peu de temps à l'examen du code de la route. Thème par thème, vous évoluerez progressivement pour maitriser le code de la route. La formation se déroule en alternant les cours et les tests afin de mieux vous perfectionner. Vous passerez ensuite un examen de code blanc qui confirmera vos acquis avant de passer l'examen final dans les jours qui suivent. Si toutefois l'examen blanc n'a pas été favorable, vous pourrez continuer à compléter votre formation gratuitement en salle de code. Des cours de code peuvent en fonction de votre situation, être mis en place en distanciel.

Cette formule comprend :

  • La Formation théorique
  • livret Code
  • 1 Présentation à l'examen du Code de la Route
  • 1 Accompagnement à l'examen de Conduite
  • 1 Pack Web accès illimité au Code
